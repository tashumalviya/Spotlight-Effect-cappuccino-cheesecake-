# Spotlight-Effect(cappuccino-cheesecake)
Spotlight Effect

Interactive spotlight effect with a cozy coffee & cheesecake theme! Move your mouse and watch the spotlight follow your cursor.  

## Features
- Full-screen coffee & pastry background  
- Spotlight follows mouse  
- Hidden cursor for cinematic vibe  
- Fun glowing text: *“Espresso Yourself… Cheesecake Style!!😋”*
